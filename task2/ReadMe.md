
## Создать приложения:

#### Приложение 1. Я и моя образовательная программа. Получение, обработка и представление данных . <br>Содержание приложения: 

1. Создание моделей о себе, руководстве, сокурсниках содержание из контрольной 1
2. Полученные из формы  отзывов об образовательной программе (ник комментатора, содержание, оценка по десятибальной шкале, дата отправки отзыва ) ); 
3. Cохранение данных в таблицах базы, 
4. Получение данных из таблицы базы,  представление данных  и результатов в таблицеподобном виде,
5. Фильтрация и сортировка данных на примере одной из таблиц
6. Представление в агрегированном виде статистики  по одному из полей таблицы
 
Все страницы приложения объединить единой навигацией.
